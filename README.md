# ☕ Coffee Bean Sorting System

A computer vision project for classifying and sorting coffee beans using deep learning techniques. This system leverages object detection models to identify and categorize coffee beans efficiently.

## 🚀 Overview

This project focuses on solving a real-world agricultural problem: **automated coffee bean classification**.  
The system was initially developed using PyTorch and later optimized by transitioning to YOLO for real-time object detection.

## 🛠 Tech Stack

- **Deep Learning Framework:** PyTorch  
- **Object Detection:** YOLO (You Only Look Once)  
- **Language:** Python  
- **Tools:** OpenCV, NumPy  
- **Annotation Tool:** Roboflow  

## ✨ Features

- Detect and classify coffee beans using computer vision
- Real-time object detection using YOLO
- Improved performance compared to initial PyTorch model
- Scalable pipeline for training and inference
- Practical application in agricultural automation

## 🧠 Project Workflow

1. Data Collection  
   - Collected and prepared dataset of coffee beans  

2. Data Annotation  
   - Labeled images for training object detection model  

3. Model Development  
   - Built initial model using PyTorch  
   - Trained and evaluated classification performance  

4. Model Optimization  
   - Migrated to YOLO for better speed and accuracy  
   - Implemented real-time detection pipeline  

5. Evaluation  
   - Tested model performance on unseen data
